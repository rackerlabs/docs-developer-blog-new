Getting Started with Node.js SDK

Rackspace's official SDK for Node.js pkgcloud, an MIT licensed open-source community project for using multiple cloud providers from a standardized API.

Currently, pkgcloud offers support for many Rackspace services:

  • CDN
  • Cloud Block Storage
  • Cloud DNS
  • Cloud Files
  • Cloud Load Balancers
  • Cloud Servers
  • Cloud Orchestration

Support for additional services is on-going.

Requirements

Using pkgcloud requires Node.js 0.8 or newer and npm 1.3 or newer. You can find installers and source on nodejs.org; npm is bundled with Node.js installers.

Installation

Generally, all Node.js packages are recommended to be installed via npm, and pkgcloud is no different:

{% highlight sh %} $ npm install pkgcloud{% endhighlight %}

Using pkgcloud

For any combination of service & region, a user should create a single client, and then use that client to access APIs for that service & region combination. If you had assets in both ORD & IAD you would create two clients.

In this example, we create a single compute client, in the IAD region, and then output a list of server Id's to the console:

{% highlight javascript %} var pkgcloud = require('pkgcloud'); var client = pkgcloud.providers.rackspace.compute.createClient({ username: 'my-username', apiKey: 'my-api-key', service: 'compute', region: 'IAD' }); client.getServers(function(err, servers) { if (err) { // handle your error case } // print out each server id servers.forEach(function(server) { console.log(server.id); }); });{% endhighlight %}

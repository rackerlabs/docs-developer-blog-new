Getting Started with the PHP SDK

Rackspace's official PHP SDK is php-opencloud, an open-source PHP library released under the Apache 2.0 license.

php-opencloud supports both Rackspace and OpenStack services, such as:

  • Auto Scale
  • CDN
  • Cloud Block Storage
  • Cloud Databases
  • Cloud DNS
  • Cloud Files
  • Cloud Images
  • Cloud Load Balancers
  • Cloud Monitoring
  • Cloud Queues
  • Cloud Servers
  • Rackspace Identity
  • Cloud Orchestration

Requirements

To install the SDK, you will need to have Composer installed first. There are install guides for both Linux/OSX and Windows.

Installation

Once you have installed Composer, you will need to run:

{% highlight sh %} $ composer require rackspace/php-opencloud:dev-master {% endhighlight %}

This will install the SDK and its dependencies to the `vendor` folder in your current directory.

Using php-opencloud

Before consuming API services you will need to pass in your authentication details when creating a client. Authentication is handled for you automatically:

{% highlight php %} '{username}', 'apiKey' => '{apiKey}' )); {% endhighlight %}

After this initial setup, you can then interact with an API service. In this example, we'll create a new Cloud Files container and upload some files:

{% highlight php startinline %} $objectStoreService = $client->objectStoreService(null, '{region}'); $container = $objectStoreService->createContainer('logos'); $localFileName = '/path/to/local/php-elephant.jpg'; $remoteFileName = 'php-elephant.jpg'; $handle = fopen($localFileName, 'r'); $container->uploadObject($remoteFileName, $handle); fclose($handle); {% endhighlight %}

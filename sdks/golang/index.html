Getting Started with the Go SDK

Rackspace's official SDK for Go is Gophercloud, an Apache-licensed open-source community project for using multiple cloud providers from a standardized API.

Currently, Gophercloud offers support for multiple Rackspace services:

  • CDN
  • Cloud Block Storage
  • Cloud Files
  • Cloud Load Balancers
  • Cloud Servers
  • Cloud Orchestration

Requirements

  1. The Go Programming Language
  2. Git

Installation

Install Gophercloud by executing the following command (after you have installed Go): {% highlight bash %} go get github.com/rackspace/gophercloud {% endhighlight %}

Using Gophercloud

To interact with Rackspace using Gophercloud, you'll first need to create an authenticated Rackspace session, represented by a provider client. Then, you can use the provider client to obtain a service client to interact with a specific cloud service in a given region. The service client can be given to the request functions within that service's resource packages to interact with the service.

In this example, we'll interact with the object storage service:

{% highlight go %} import ( "github.com/rackspace/gophercloud" "github.com/rackspace/gophercloud/rackspace" "github.com/rackspace/gophercloud/rackspace/objectstorage/v1/accounts" ) provider, err := rackspace.AuthenticatedClient(gophercloud.AuthOptions{ Username: "{username}", APIKey: "{apiKey}", }) service, err := rackspace.NewObjectStorageV1(provider, gophercloud.EndpointOpts{ Region: "{region}", }) metadata, err := accounts.Get(service).ExtractMetadata() {% endhighlight %}

Go

Requirements

  1. Go
  2. Git

Install Gophercloud, the Go SDK for Rackspace

Run the following command (after installing Go):

{% highlight bash %} go get github.com/rackspace/gophercloud {% endhighlight %}

Then use it in your code by importing it:

{% highlight go %} import ( "github.com/rackspace/gophercloud" "github.com/rackspace/gophercloud/rackspace" ) {% endhighlight %}

Java

Requirements

  1. Java Development Kit (JDK) version 6 or higher
  2. Maven version 3 or higher

Install JClouds, the Java SDK for Rackspace

Download this pom.xml file.

{% highlight bash %} mvn dependency:copy-dependencies "-DoutputDirectory=./lib" {% endhighlight %}

Tip: To make JClouds a part of your existing project, simply copy the relevant dependencies into your own pom.xml file.

.NET

Install Rackspace.NET, the .NET SDK for Rackspace

{% highlight bash %} PM> Install-Package openstack.net {% endhighlight %} {% highlight csharp %} using net.openstack.Providers.Rackspace; using net.openstack.Core.Providers; using net.openstack.Core.Exceptions.Response; {% endhighlight %}

Node.js

Requirements

  1. Node.js
  2. NPM

Install pkgcloud, the Node.js SDK for Rackspace

{% highlight bash %} npm install pkgcloud {% endhighlight %} {% highlight js %} var pkgcloud = require('pkgcloud'); var client = pkgcloud.providers.rackspace.createClient({/* ... */}); {% endhighlight %}
